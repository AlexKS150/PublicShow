  • 不只是同仁受惠，同仁子女也有！

    好處證據超連結

    補習班災難會在不同人且不同時期發生，以國中補習班為例，看起來非常順利，實際上已經是多花金錢跟時間在不必要的地方，成績好不代表真的需要補習班去維持，甚至有些人的子女送進補習班但還是越補越不行，被補習班認為這學生沒有好好認真讀書，花了很多錢在補習班，而且又怕市售上的自修參考書會比不上補習班的資源，雖然份量都是差不多一樣，但是難度的話市售的參考書就是沒辦法比上補習班難度，我這裡可是有A級到C級的難度都有，甚至同一等級有6份多，五科都有，而且有中國的資優題目可以讓學生有額外的時間去多多練習，所以國中的學生都不用怕題目不夠又沒難度，要說高中的學生呢？我則是有更多高中的資源且有更多題目讓他們多多練習，都是明星高中的老師的部落格下載下來的，所以難度會達標。小學的話我也有小學的數學跟英語以及國語文讓他們準備下一階段，大學不用講，我已經有了共同必修科目的資料，微積分、線性代數、基礎物理跟基礎化學還有工程數學讓他們在暑假好好自修，如果知識點都已經熟悉透了就不必花額外的金錢在書籍上了（除非像我真的非常需要書籍搭起我的教育服務網站），我的目的就是降低教育費用，從小學到升高中和升大學考試的補習課外教育都要降低費用，讓學生有自主學習精神之外而且又降低了15萬在課外教育費用上。
