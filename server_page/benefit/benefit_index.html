  • 不只是同仁受惠，同仁子女也有！

    補習班災難不只有某個期間跟地點，所以更需要擁有額外的幫助，每個同仁的子女就是有不一樣的狀況，同一間補習班還是會有不同效果，就怕子女已經非常努力讀書還是考不好，不是小孩子不努力，多半是補習班老師的素質非常差，都已經放學還要花更多時間在補習班老師的身上，如果補習班老師卻還是有教育偏差，就算是有開資優班，還是比不上在家已經學好東西且在段考得到好成績。可是自學的資源又是少之又少，而且補習班老師不可能願意多給學生更多資源，因為都是賺錢工具，怕給太多會失去賺錢機會，甚至被同業擊敗。我可以幫忙，但是不要濫用，例如：需要一學期的英文講義。
