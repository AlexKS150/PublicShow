台灣旅遊推薦(Vue工具)

專案型設計

作品介紹

這是個人學Vue時，所設計出來的作品，首頁也兩區Swiper，第一個會自動播放，而且手動切換後不會中斷，第二個是飯店跟旅行社連結區，可以連結到該網址，旅遊推薦是本作品的精華，在左上角按出縣市名稱，按出想要縣市後，立即出現該縣市不同旅遊景點推薦，按查看詳情可以看到景點圖片已經地圖，重點是我沒有用過Google Map Javascript API，這裡使用簡單HTML5格式即可，回上一頁返回到原本位置而不是到頂部。
