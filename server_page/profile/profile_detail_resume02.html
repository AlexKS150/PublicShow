專用前端網頁製作工具

前端開發工具梁狼滿目，但是會挑選「國際上前端工程師都會用到的工具」，而不是把整個工具都學會，在自學當中，我得到的答案是，設計前端需要測量網站規格大小，我是用Mark Man，免費，因為PhotoShop我並沒有許用很多，而且太多工具我沒用到，Mark Man是簡潔的選擇，在網站生產自動化方面，國際上都推薦Gulp跟Webpack，雖然兩個都很好，但是自學會強調先把一個東西弄熟就好，不然工作中還要懷疑目前的工具不好用還要換另外一個工具會有很大的時間成本，企業要求的是穩定不是改變創新，我就立即判斷下選擇Gulp,我在Gulp學得還可以就使用了。前端進階工具我則是選擇Vue，並不是主流或者是很都課程都開Vue所以去學，而是Vue的社群開發維護比較頻繁，甚至有例子導覽可以看，更何況有更多教學影片可以去看，不只如此，Vue的開發網站的觀念是我比較能掌握，Vue檔裡的三大模塊Template、Script和Style，正式HTMl5、Javascript跟CSS3的觀念非常相似，所以用Vue。至於CSS設計方面，我有用過Sass跟Stylue，目前這一件是用Sass，而台灣旅遊推薦適用Stylue。我比較偏好Sass。Bootstrap插件我有使用過，第三件小型設計就是使用Bootstrap 3。
